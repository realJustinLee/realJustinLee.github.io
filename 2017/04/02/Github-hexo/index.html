使用GitHub和HEXO搭建你的个人博客

为什么使用GitHub和HEXO？

  • GitHub是全球最大的程序员交友网站，在这里写博客逼格自然没得说
  • HEXO是一款基于Node.js的静态博客框架，文章以Markdown格式书写保存，让您不在纠结于数据库的增删改查。强大的可扩展性、丰富的插件和主题，让您的博客真正地成为您独一无二的博客
  • 本文默认您使用Shell作为您创建本博客的工具

主要步骤

首先，您得有个Github账号

  • 注册GitHub作为程序员的基本功，这里就不再赘述了

之后，您需要给您的博客安个家

  • 创建一个名为<您的GitHub用户名>.github.io的仓库
  • 建仓库的方法本文就不赘述了，如果您没有这方面的经验，请点击这里

很好！如果您看到这里，想必您的仓库已经在GitHub中拥有了一席之地，那我们继续吧！

  • 克隆你的仓库到本地
    ~ $ git clone https://github.com/username/username.github.io

下面我们来安装HEXO

  • 点击这里访问HEXO官网，请务必收藏它。因为在您搭建博客遇到难题时，此网站会给予您莫大的帮助

    这里，我们采用nvm进行Node.js的安装及版本管理

  • 安装nvm
    ~ $ curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.33.8/install.sh | bash
  • 安装最新稳定版本的Node.js
    ~ $ nvm install stable
  • 安装HEXO
    ~ $ npm install hexo-cli -g

好了，前期的准备工作已经完成，我们来建立博客吧

  • 部署HEXO，并且书写第一篇文章，左后生成静态页面
  • 注意：每次修改博客都需要重新生成博客静态页
    ~ $ mkdir <您的博客文件夹的名字>
~ $ cd <您的博客文件夹的名字>
~ $ hexo init
~ $ hexo new hello-world
~ $ vim source/_posts/hello-world.md
~ $ hexo g
  • 预览博客，即启动HEXO本地服务器，在正式发布之前，您可以在本地环境中测试您的博客
    ~ $ hexo s

    当一切大功告成，我们正式上线我们的博客

    ~ $ hexo d
